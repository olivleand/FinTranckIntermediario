# FinTrack

Sistema de controle de finanças pessoais desenvolvido em Java.

## Funcionalidades

- Cadastro de receitas e despesas
- Atualização de transações
- Remoção de transações
- Listagem em tabela
- Persistência com SQLite

## Tecnologias

- Java
- JavaFX
- FXML
- CSS
- JDBC
- SQLite
- JUnit
