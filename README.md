# 💰 FinTrack Intermediário

Sistema de controle de finanças pessoais desenvolvido em Java como projeto do módulo intermediário do curso de Desenvolvimento Java.

## 📋 Descrição

O FinTrack permite o gerenciamento de receitas e despesas por meio de uma interface gráfica desenvolvida com JavaFX. As informações são armazenadas em banco de dados SQLite, possibilitando o cadastro, consulta, atualização e remoção de transações financeiras.

## 🚀 Funcionalidades

* Cadastro de receitas e despesas
* Listagem de transações em tabela
* Atualização de registros
* Remoção de transações
* Cálculo de saldo financeiro
* Persistência de dados com SQLite
* Interface gráfica desenvolvida com JavaFX

## 🛠 Tecnologias Utilizadas

* Java
* JavaFX
* FXML
* CSS
* JDBC
* SQLite
* JUnit 5
